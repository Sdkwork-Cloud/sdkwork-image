# Plugins

Application-local agent extensions for `sdkwork-image-pc`.

## How to Add Plugins

1. Create a directory: `.sdkwork/plugins/<plugin-name>/`
2. Add `.codex-plugin/plugin.json` for installable plugins
3. Optionally add `skills/`, `mcp/`, `apps/`, `scripts/`
4. Plugin names must be lowercase kebab-case

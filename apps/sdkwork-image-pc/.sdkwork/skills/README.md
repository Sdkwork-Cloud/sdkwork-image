# Skills

Application-local agent workflows for `sdkwork-image-pc`.

## How to Add Skills

1. Create a directory: `.sdkwork/skills/<skill-name>/`
2. Add `SKILL.md` as the entrypoint
3. Optionally add `references/`, `scripts/`, `assets/`
4. Skill names must be lowercase kebab-case
5. Skills must cite relevant root specs instead of copying them

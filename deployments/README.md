# Deployments

Deployment descriptors, environment topology, and release documentation.

## Purpose

Stores deployment topology, infrastructure descriptors, release handoff files, and deployment runbooks.

## Owner

Repository maintainers.

## Allowed Content

- Docker files
- Kubernetes manifests
- Systemd units
- Nginx configs
- Deployment runbooks

## Forbidden Content

- Live secrets or private keys
- Local override files
- Runtime user config

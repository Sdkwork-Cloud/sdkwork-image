# Scripts

Thin command entrypoints for build, verification, generation, and release workflows.

## Purpose

Stores shell/Node/Python/PowerShell command wrappers.

## Owner

Repository maintainers.

## Allowed Content

- Build scripts
- Verification scripts
- Generation scripts
- Migration scripts
- Release scripts

## Forbidden Content

- Reusable logic (belong in `tools/` or proper packages)
- Secrets or credentials

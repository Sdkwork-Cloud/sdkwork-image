# Configs

Source-controlled safe config templates, profile examples, and config schemas.

## Purpose

Stores config templates, schemas, profile examples, and non-secret runtime defaults.

## Owner

Repository maintainers.

## Allowed Content

- Config templates
- Config schemas
- Profile examples
- Non-secret defaults

## Forbidden Content

- Live secrets or credentials
- User-private runtime config
- Runtime state

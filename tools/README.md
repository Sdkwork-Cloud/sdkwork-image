# Tools

Developer, validation, generation, migration, and operator tools.

## Purpose

Stores reusable validators, generators, migration tools, parsers, CLIs, and operator utilities.

## Owner

Repository maintainers.

## Allowed Content

- Validators
- Generators
- Migration tools
- Parsers
- CLIs
- Operator utilities

## Forbidden Content

- Application runtime code
- Generated SDK output
- Secrets or credentials

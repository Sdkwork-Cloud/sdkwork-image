# Plugins

Application/runtime plugin source packages and extension packages.

## Purpose

Stores application/runtime installable extension source.

## Owner

Repository maintainers.

## Allowed Content

- Plugin source packages with `specs/component.spec.json`
- Plugin src/ and tests/

## Forbidden Content

- Agent plugin workspaces (belong in `.sdkwork/plugins/`)
- Generated SDK output
- Secrets or credentials

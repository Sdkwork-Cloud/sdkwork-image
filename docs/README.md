# Documentation

Repository/application documentation, architecture decisions, and runbooks.

## Purpose

Stores documentation beyond root README files.

## Owner

Repository maintainers.

## Allowed Content

- Architecture decisions (ADR)
- Runbooks
- Changelogs
- User/developer guides

## Forbidden Content

- Secrets or credentials
- Generated SDK output
